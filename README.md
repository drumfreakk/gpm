# gpm


## Compiling
First, set USER_AGENT in cmake or CMakeLists.txt to a user agent you like, something like ```username-gpm``` or something simmilar. Then, execute these commands:

	$ mkdir build && cd build
	$ cmake ..
	$ make
And run the new executable.
